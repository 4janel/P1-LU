Proyecto 1

Crear una función que tome un objeto y devuelva las keys y values como matrices separadas. Devuelve las keys ordenadas alfabéticamente, y sus correspondientes values en el mismo orden. 

keysAndValues({ a: 1, b: 2, c: 3 }); 
→ [["a", "b", "c"], [1, 2, 3]]
keysAndValues({ a: "Apple", b: "Microsoft", c: "Google" });
→ [["a", "b", "c"], ["Apple", "Google", "Microsoft"]]
keysAndValues({ key1: true, key2: false, key3: undefined });
→ [["key1", "key2", "key3"], [true, false, undefined]]

Resolución:


            function keysAndValues(obj) {
                let claves = [];
                let valores = [];
                for (let elemento in obj) {
                  claves.push(elemento);
                  valores.push(obj[elemento]);
                }
              
                let resultado = [claves.sort(), valores.sort()];
                console.log({resultado});
              }
            
            keysAndValues({ a: 1, b: 2, c: 3 });
            keysAndValues({ a: "Apple", b: "Microsoft", c: "Google" });
            keysAndValues({ key1: true, key2: false, key3: undefined });
